# wsl-my_vue_start

## 利用vue-cli脚手架构建项目的步骤如下
- 下载安装**vue-cli**
```shell
npm install -g @vue/cli
```
- 安装**vue-cli**成功之后，执行以下命令，会显示**vue**的版本号
```shell
vue --version
```
- 创建新项目
    - 通过**命令**创建项目
    ```shell
    vue create hello-world
    ```
    - 通过**图形化界面**创建项目,执行下边的命令之后，就会弹出一个图形化界面
    ```shell
    vue ui
    ```
- [vue/cli的详细用法的链接](https://cli.vuejs.org/zh/guide/)
